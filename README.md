# Food Ordering Web App
A responsive Food Ordering Web Application inspired by kiosk-style ordering systems of Mcdonald's.

## Features
- Login & Signup with localStorage
- Add to cart and quantity controls
- Automatic total price calculation
- Responsive layout for mobile and desktop
- Live order summary list

## Tech Stack
- HTML
- CSS
- JavaScript
